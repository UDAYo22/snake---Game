# Classic Snake Game

A simple, interactive Snake Game built using vanilla HTML, CSS, and JavaScript.

## Features
* Classic arcade gameplay.
* Responsive keyboard controls (Arrow keys).
* Score tracking.

## How to Play
1. Open the game in your browser.
2. Use the arrow keys to control the direction of the snake.
3. Eat the food to grow and increase your score.
4. Avoid crashing into the walls or your own tail!

## Technologies Used
* HTML5
* CSS3
* JavaScript (ES6)
